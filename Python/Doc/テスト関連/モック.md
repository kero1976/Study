# モックのネスト
## やりたいこと
モックAのあるメソッドがモックBを返し、モックBのあるメソッドの結果が必要。
普通にやると
mockB = MagicMock()
mockB.bar.return_value = "これが必用"
mockA = MagicMock()
mockA.foo.return_value = mockB
## 解決方法
mock =MagicMock()
mock.foo.return_value.bar.return_value = "これが必用"
