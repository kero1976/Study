# 4.テストの実行

## 4-1.ログ出力を有効化
--log-cli-level=DEBUG

### 4-1-1.特定のクラスログを制御
テストファイルの中で、以下の用に特定のクラスのログレベルを上げることにより実現できた

    botocore = logging.getLogger("botocore")
    botocore.setLevel(logging.ERROR)
    urllib3 = logging.getLogger("urllib3")
    urllib3.setLevel(logging.ERROR)
