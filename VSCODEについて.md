# 設定

## 日本語化

### 言語パックのインストール

1. 「Extensions」を選択しする
2. 検索欄に「Jap」と入力すれば「Japanese Language Pack VS Code」が表示されるのでインストールする
3. 再起動する

### 言語の切り替え

1. コマンドパレットを表示する
2. displayを入力する
3. 「Configure Display Language」を選択する
4. 言語が表示されるので、日本語を選択
