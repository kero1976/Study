# 1.Black Formatter

## 1-1.設定

```
{
    "[python]": {
        "editor.defaultFormatter": "ms-python.black-formatter",
        "editor.formatOnSave": true
      }
}
```

* 保存時に自動でフォーマット

## 1-2.flake8との競合回避

### 1-2-1.E208
:の前後に空白があるとエラー

```
{
    "[python]": {
        "editor.defaultFormatter": "ms-python.black-formatter",
        "editor.formatOnSave": true
      },
      "flake8.args": [
        "--max-line-length=88",
        "--ignore=E203,W503,W504"
    ]
}
```
